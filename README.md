# Weather App

A simple and responsive **Weather Application** built with **HTML**, **CSS**, and **JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

## Features
- Search for any city and get real-time weather information
- Display temperature, humidity, and wind speed
- Show weather icons dynamically based on conditions
- Handle invalid city names with error messages
- Responsive and user-friendly design

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
